# Research Card: Wampanoag Dictionary — GitHub Open-Source Project

## 1. Source Description

**Project:** wampanoag-dictionary/dictionary — an open-source web application for the Wampanoag language.

**Platform:** GitHub (github.com/wampanoag-dictionary/dictionary).

**Tech stack:** React + Express (JavaScript/Node.js).

**Features:** Search functionality for words in both English and Wôpanâak, providing translations, meanings, and example sentences. Includes historical information about the Wampanoag language and the language reclamation project.

## 2. Dialect Affiliation

Wampanoag (Wôpanâak) — the modern reconstructed form of the language, based on the WLRP's work.

## 3. Data Source

The dictionary data appears to be derived from the WLRP's modern Wôpanâak dictionary. The exact relationship between this project and the official WLRP dictionary is not documented in the repository.
