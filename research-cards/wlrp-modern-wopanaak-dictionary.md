# Research Card: WLRP — Wôpanâak Language Reclamation Project — Modern Wampanoag Dictionary

## 1. Source Description

**Project:** Wôpanâak Language Reclamation Project (WLRP), founded 1993.

**Founder/Director:** Jessie "Little Doe" Baird (b. 1963) — linguist, Algonquianist, MacArthur Fellow (2010). Earned a Master's degree in Algonquian Linguistics from MIT (2000), where she studied under Kenneth Hale and Norvin Richards.

**Scope:** A comprehensive modern dictionary of the Wampanoag (Wôpanâak) language, reconstructed from historical sources. The dictionary now contains over 10,000–13,000 words, with ongoing expansion.

**Governance:** Inter-tribal non-profit (Wôpanâak Language and Cultural Weety8, Inc.) serving the four remaining Wampanoag tribes: Mashpee Wampanoag Tribe, Aquinnah Wampanoag Tribe, Assonet Band of Wampanoag, and Herring Pond Wampanoag Tribe.

## 2. Dialect Affiliation

Wôpanâak (Wampanoag) — specifically the Martha's Vineyard and coastal Massachusetts varieties. The modern orthography is a phonemic system developed by Baird based on her analysis of the historical sources (primarily Eliot's orthography and the John Cotton Jr. vocabulary).

## 3. Reconstruction Methodology

Baird's methodology is documented in her MIT thesis and subsequent publications:

1. **Primary sources:** Eliot's Massachusett Bible (1663), Eliot's *Indian Grammar Begun* (1666), John Cotton Jr.'s vocabulary (1666–1678), Josiah Cotton's vocabulary (1707–1708), and other colonial sources
2. **Comparative Algonquian:** Uses Proto-Algonquian reconstructions (Aubin 1975, etc.) to resolve ambiguities in the colonial orthography
3. **Modern phonemic orthography:** Developed a practical writing system distinct from Eliot's orthography, representing the reconstructed phonemic inventory
4. **Community review:** Dictionary entries are reviewed by the WLRP language committee and community elders

## 4. Orthographic System

The modern Wôpanâak orthography is a phonemic system using standard Roman letters with some diacritics:

- Vowels: a, â, e, i, o, u (with length marked by acute accent or other conventions)
- Consonants: p, t, k, s, š, č, h, m, n, w, y
- Preaspiration: written as h before stops (hp, ht, hk, hč)
- The symbol ∞ (Eliot's ⟨ꝏ⟩) is represented differently in the modern system

This is distinct from Eliot's 17th-century orthography and represents a phonemic analysis of the language.

## 5. Accessibility Status

- **WLRP website:** Publicly accessible at wlrp.org — includes basic vocabulary and cultural information
- **Full dictionary:** Not publicly available online as a downloadable resource. Used in WLRP classes and materials.
- **Published materials:** WLRP has produced textbooks, children's books, and teaching materials, but these are primarily distributed within the Wampanoag community.
- **MIT page:** Norvin Richards' page (web.mit.edu/norvin/www/wopanaak.html) provides some linguistic information.

## 7. Key Features

- **Largest modern Wampanoag dictionary** — 10,000+ entries
- **Phonemic orthography** — provides IPA-accessible forms for comparison with colonial spellings
- **Community-based** — developed with and for the Wampanoag community
- **Comparative methodology** — uses PA reconstructions to resolve colonial orthographic ambiguities
- **Living language** — WLRP has produced fluent speakers, making this a living language resource

## 8. Utility as Comparison Baseline

The WLRP dictionary is arguably the single most important comparison source for the shoebox editor project because:

1. It provides a **phonemically analyzed** form for each word, which can be compared with the ad-hoc colonial spellings
2. It represents the **same language** (Wampanoag) as the primary colonial sources
3. The reconstruction methodology is **documented and transparent** (Baird's MIT thesis)
4. It can serve as a **validation baseline** — if a colonial spelling maps cleanly to the WLRP phonemic form, the colonial source is reliable; if not, the discrepancy needs analysis

## 9. Research Gaps

- The full dictionary is not publicly accessible as a downloadable dataset
- The mapping between WLRP's modern orthography and Eliot's orthography is not fully documented in a publicly available reference
- Systematic comparison of WLRP forms with the colonial sources used in #1362–#1369 has not been done
- The WLRP dictionary's coverage of specialized vocabulary (kinship, material culture, etc.) vs. basic vocabulary needs assessment
